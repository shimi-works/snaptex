# SnapTeX ― 数式スクショ → LaTeX

PDFやWebのスクショに写った**数式を認識して LaTeX テキストに書き起こす**、単一HTMLのツール。
画像をそのまま貼るのではなく「文字」で送れるようにして、ChatGPTなどの画像制限を回避するのが目的。

- アプリを開く … `index.html`（GitHub Pages: /snaptex/）
- 技術説明・設計思想 … `portfolio.html`（/snaptex/portfolio.html）

## 使い方
1. ⚙設定から **Gemini APIキー（無料・クレカ不要）** を入力（[取得はこちら](https://aistudio.google.com/app/apikey)）
2. 数式のスクショを用意
   - **PC**: `Win`+`Shift`+`S` で範囲スクショ → アプリ上で `Ctrl`+`V`
   - **スマホ**: 画面をタップして、スクショ画像を選択（カメラ撮影も可）
3. **「LaTeXに変換」** → 出てきた LaTeX をコピーして貼り付け

## 特徴
- 単一HTML・ノーサーバー・**PC / スマホ両対応**
- Gemini API をブラウザから直接呼び出し（各自の鍵を入力＝BYOK）
- 出力2モード:「**文章＋数式**」（本文＋ `$…$` / `$$…$$`・ChatGPT貼付向き）と「**数式だけ**」
- **区切り記号トグル**（`$付き` ⇔ `なし=Word数式向け`）を生成しなおさず切替
- **式ごとコピー**：複数式は1式ずつコピーボタン付きで並び、Wordの数式ボックスへ個別に貼れる
- MathJax による数式プレビューで認識結果を目視確認、ズレたら手直し
- 履歴（localStorage）・ワンクリックコピー
- 無料枠対策（上限で自動モデル切替・自動リトライ）

## プライバシー
- APIキーはこの端末の localStorage にのみ保存。Google 以外へは送りません。
- 画像は変換時に Google (Gemini) へ送信されますが、アプリ側には保存しません。
- 共有PCでは設定からキーを消去してください。
