SnapTeX

スクショの数式を LaTeX テキストに ― ChatGPTへ文字で送るために 技術

① 画像を渡す

ここに Ctrl+V で貼り付け
Win+Shift+S で範囲スクショ → そのまま貼り付け
／ 画像ファイルをドロップ・選択（複数まとめてOK）
📷 タップして画像を選択
スマホのスクショ（数式の画面）を選んでください
複数選択・カメラ撮影もOK
出力：
「文章＋数式」＝画像の中身をすべて書き起こし、数式は $…$／$$…$$ に。ChatGPTへの貼り付け向き。化学式・反応式も \ce{…}（mhchem）で出力します。

履歴（この端末に保存）

まだありません。

② 結果（LaTeXテキスト）

区切り記号：
Word用：数式ボックス内では日本語が斜体になるため、「数式だけ」モードをおすすめします。Alt+= の数式ボックスに貼り（複数式まとめてOK）、数式タブで {} LaTeX をONにして 変換▼→「すべて - 2次元形式に変換」を選ぶと全式を一括変換できます。

式ごとにコピー ― 複数式のとき。1式ずつWordの数式ボックスへ

③ 数式プレビュー（確認用）

プレビューは MathJax（ネット接続時）で描画。ズレていたら結果テキストを直接直せます。
キーはこの端末の localStorage にのみ保存。画像は変換時に Google(Gemini) へ送信されます・保存はされません。共有PCでは設定からキーを消してください。

設定

このアプリは Google の Gemini（無料枠・クレカ不要） を直接呼び出します。キーはこの端末にのみ保存され、Google以外へは送りません。

取得：Google AI Studio → Get API key（無料）
